# PigWxWork
#### [English Readme](https://github.com/PhongSeow/PigWxWork/blob/master/README.md)
这是一个接口简单，容易理解的企业微信开发工具应用，现在刚刚起步，后续还会不断迭代更多功能。

## ***目录和文件描述***

### Release

发布执行码目录，如果你不想看到源程序，你可以直接使用这个目录中的文件。

##### Release\DotNet\PigWxWorkLib
类库的DLL文件和 NuGet 包。

### Src

源码目录。

#### Src\DotNet\PigWxWorkLib

类库目录

#### Src\DotNet\PigWxWorkDemo

示例目录
