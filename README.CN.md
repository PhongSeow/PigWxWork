# ObjAdoDB
#### [English Readme](https://github.com/PhongSeow/ObjAdoDB/blob/main/README.md)
这是一个将 Microsoft ActiveX Data Objects Library 映射到 .net 平台的类库和 NuGet 包，可以在不同的 .net 框架下使用。
类库需要 %ProgramFiles(x86)%\Common Files\System\ado\msado15.dll 支持，Windows 通常默认已安装好了。
使用这个类库，可以方便将 VB6 和 ASP 的代码升级到 .net 平台。

## ***目录和文件描述***

### Release

发布执行码目录，如果你不想看到源程序，你可以直接使用这个目录中的文件。

##### Release\DotNet\ObjAdoDBLib
类库的DLL文件和 NuGet 包。

### Src

源码目录。

#### Src\DotNet\ObjAdoDBLib

类库目录

#### Src\DotNet\ObjAdoDBDemo

示例目录
